Предназначен для выбора присоединенных файлов. Добавить файл в списке выбора невозможно. Для добавления файлов перейдите в список Присоединенные файлы.

Вызывается по кнопке  Выбрать из объектов программы, в которых предусмотрен выбор присоединенных файлов.

В списке отображаются наименования файлов, значки перед наименованием указывают на тип файла.

Просмотр сведений о файле

  • С помощью кнопки  Свойства файла (клавиши F2) можно открыть карточку присоединенного файла и просмотреть сведения о нем.

Выбор файла

  • Выделите файл, нажмите Выбрать. Также можно воспользоваться двойным щелчком левой кнопки мыши.
