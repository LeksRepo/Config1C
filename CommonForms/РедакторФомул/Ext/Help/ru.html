Переменные Каталог

 

ШиринаИзделия - ширина изделия по каталогу задается в форме каталога.

ВысотаИзделия - высота изделия по каталогу задается в форме каталога.

ГлубинаИзделия - глубина изделия по каталогу задается в форме каталога.

 

Столешница - высота столешницы задается в форме каталога (26, 38, 40).

Европаз - деталь зависит от значения европаза (8, 16).

 

ПрипускСверху - переменная для шкафа купе, припуск у элементов сверху.

ПрипускСлева - переменная для шкафа купе, припуск у элементов слева.

ПрипускСправа - переменная для шкафа купе, припуск у элементов справа.

ПрипускСнизу - переменная для шкафа купе, припуск у элементов снизу.

 

ЗавалПол - переменная для шкафа купе, завал на полу.

ЗавалЛевый - переменная для шкафа купе, завал стены слева.

ЗавалПравый - переменная для шкафа купе, завал стены справа.

ЗавалПотолок - переменная для шкафа купе, завал потолка.

ЗавалСтенка - переменная для шкафа купе, завал задней стенки.

 

Отсек{НомерОтсека}  - Ширина отсека. Например: Отсек1.

ВысотаПроема - Высота проема двери шкафа по каталогу.

ШиринаПроема - Ширина проема двери шкафа по каталогу.

ВысотаПотолок - Высота потолка шкафа по каталогу.

ВысотаПол - Высота пола шкафа купе по каталогу.

ШиринаПравый - Ширина правого элемента шкафа по каталогу.

ШиринаЛевый - Ширина левого элемента шкафа по каталогу.

Раздвижная - Вид двери(Да или нет).

Распашная - Вид двери(Да или нет).

РадиусЛ - Радиус справа.

РадиусП - Радиус слева.

ГлубинаМатериала - Тольщина основного материала.

ГлубинаМатериалаДоп - Тольщина доп материала.

 

Возможно обращение к свойствам выбранной номенклатуры например:

 

НоменклатураМатериала.ДлинаДетали - длина детали (например лдсп 16: 2720).

НоменклатураМатериала.ШиринаДетали - ширина детали (например лдсп 16: 1800).

НоменклатураМатериала.ГлубинаДетали - глубина детали (например лдсп 16: 16).

 

Возможно использование встроенных функций 1С:

 

Макс(<Значение1>,...,<ЗначениеN>) - Определяет максимальное значение из полученных параметров.

Мин(<Значение1>,...,<ЗначениеN>) - Определяет минимальное значение из полученных параметров.

?(Условие,Значение если условие выполнилось, значение если условие не выполнилось) - Определяет какое значение установить в зависимости от выполнения условия.
