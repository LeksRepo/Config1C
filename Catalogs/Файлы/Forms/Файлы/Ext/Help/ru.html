Предназначен для управления файлами.

В программе существует возможность хранить файлы в программе без привязки к конкретному объекту.

Список открывается по команде Файлы в разделе Органайзер. 

Работа со списком организована аналогично работе в Проводнике Windows. Левая часть списка содержит иерархическую структуру папок. В правой части располагается список файлов в выделенной папке. С помощью соответствующих команд в список можно добавлять новые файлы, копировать, изменять или удалять их.

В списке выводится:

  •  - подписан ЭП.
  •  - зашифрован.
  • Наименование - полное наименование файла. Значок рядом с наименованием указывает тип файла.
  • Описание - подробное описание файла.
  • Редактирует - пользователь, который занял файл для редактирования.
  • Версия - номер версии файла.
  • Дата - дата добавления версии файла в базу.
  • Отредактировал - пользователь, который добавил версию файла.
  • Автор - пользователь, который добавил файл.

С помощью команды Изменить форму можно добавить дополнительные поля, например:

  • Данные от - дата изменения данных версии файла (местное время).
  • Текущая версия - код текущей версии и т.д.

Зеленым цветом отображаются файлы, занятые Вами для редактирования, а серым цветом - файлом, занятые другими пользователями программы.

Управление просмотром списка файлов

С помощью кнопки  Использовать иерархию можно изменить вид списка и увидеть все файлы, находящиеся в разных папках, в одном списке.

Размещение файлов в папках

  • Файлы можно перемещать в различные папки списка с помощью перетаскивания мышью.
  • В списке можно создать необходимое количество папок и хранить в них файлы, непривязанные к конкретным объектам программы. Создать папку можно с помощью соответствующей команды списка или контекстного меню.

Права доступа к папкам

  • Папкам можно назначать различные права доступа.
    • Права доступа к папкам файлов настраиваются администратором программы или пользователем, имеющим право на управление правами той или иной папки.

Загрузка файлов и папок

  • Для загрузки файлов с жесткого диска используйте загрузку файлов или загрузку папок.
  • С помощью команды контекстного меню по правой кнопке мыши Загрузить файлы можно в одной из папок списка файлы с жесткого или сетевого диска.
  • Для того чтобы поместить папку с жесткого или сетевого диска в одну из папок списка Файлы, выполните команду контекстного меню по правой кнопке мыши Загрузить папку.

Выгрузка папок

Команда Сохранить папку (расположена в контекстном меню папки) позволяет скопировать выбранную папку с файлами на жесткий диск. При этом копируются только активные версии файлов.

Шаблоны файлов

  • В списке имеются предопределенные папки, которые нельзя удалить, например, папка Шаблоны файлов.
  • Программа поставляется с предопределенной папкой Шаблоны файлов. В папке можно разместить файлы, которые будут использоваться при создании новых файлов в качестве шаблона. Например, в эту папку можно поместить файл-заготовку с текстом договора и в дальнейшем использовать его при создании договоров для конкретных контрагентов.
  • Для того чтобы поместить в папку новый шаблон, откройте папку Шаблоны файлов, нажмите Создать, с помощью переключателя выберите Из файла на диске. Затем выберите нужный файл.
  • Программа автоматически создает и выводит на просмотр карточку файла. Заполните описание шаблона, поле Папка по умолчанию заполнено - Шаблоны файлов.
