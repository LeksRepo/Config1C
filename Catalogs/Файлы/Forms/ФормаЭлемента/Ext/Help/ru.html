Для открытия карточки файла используйте клавишу F2 или кнопку  Изменить командной панели списка файлов. Карточку можно также открыть из полнотекстового поиска или из отчетов.

Карточка файла содержит различные сведения о файле и его текущей версии и кнопки для выполнения основных действий с файлом. Дополнительные команды для работы с файлом находятся в меню Все действия командной панели формы файла.

Сведения о файле

  • Наименование - полное наименование файла. Значок рядом с наименованием указывает тип файла.
  • Описание - подробное описание файла.
  • Автор - пользователь, который добавил файл.
  • Редактирует - пользователь, который занял файл для редактирования.
  • Дата - дата добавления версии файла в базу.
  • Отредактировал - пользователь, который добавил версию файла.
  • Папка - папка, в которой сохранен файл. С помощью кнопки  можно выбрать другую папку.
  • Код - код файла.

Текущая версия

  • Номер - номер версии файла.
  • Автор - пользователь, который добавил версию файла;
  • Создана - дата добавления версии файла в базу;
  • Данные от - дата изменения данных версии файла (местное время);
  • Размер (байт) - размер данных версии файла;
  • Расширение - формат файла, рядом значок, который также показывает формат;
  • Кодировка - кодировка файла, выбирается при сохранении версии. Можно выбрать кодировку По умолчанию.

      ЭП

      На вкладке размещаются сведения об электронной подписи файла

      Зашифрован для

      • На вкладке размещаются сведения о шифровании файла.

      Из карточки файла можно перейти к просмотру и редактированию связанных с файлом сведений:

      • версии файлов;
      • задачи и бизнес-процессы.
